## Server requirements

* nodejs
* npm

## Install code

**Clone project**
```
$ git clone git@github.com:segimanzanares/react-admin-test.git
```
**Install dependencies**
```
$ cd react-admin-test
$ npm install
```
**Configure environment**<br>
Copy `.env.example` to `.env` file and edit its content.

**Run local server**
```
$ npm start
```
